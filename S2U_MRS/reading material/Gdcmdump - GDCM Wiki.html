Introduction

Tool to dump fields of a DICOM file to screen.

Doxygen

  • http://gdcm.sourceforge.net/html/gdcmdump.html

Help 

$ gdcmdump --help

gdcmdump: gdcm 2.1.0 $Date: 2008-10-17 14:21:52 +0200 (Fri, 17 Oct 2008) $
Usage: gdcmdump [OPTION]... FILE...
dumps a DICOM file, it will display the structure and values contained in the specified DICOM file
Parameter (required):
  -i --input     DICOM filename or directory
Options:
  -x --xml-dict  generate the XML dict (only private elements for now).
  -r --recursive recursive.
  -d --dump      dump value (limited use).
  -p --print     print value instead of simply dumping (default).
  -c --color     print in color.
  -C --csa       print SIEMENS CSA Header (0029,[12]0,SIEMENS CSA HEADER).
  -P --pdb       print GEMS Protocol Data Block (0025,1b,GEMS_SERS_01).
General Options:
  -V --verbose   more verbose (warning+error).
  -W --warning   print warning info.
  -D --debug     print debug info.
  -E --error     print error info.
  -h --help      print help.
  -v --version   print version.
Special Options:
  -I --ignore-errors   print even if file is corrupted.
